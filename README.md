# react-reactnative-skeleton
Skeleton to work with react / react native

# Installation

## global npm modules
	$ npm i -g react-native-cli
	$ npm i -g grunt-cli

## React
	$ cd react
	$ npm i
	$ npm start

## React Native
	$ npm i -g react-native-cli
	$ cd react-native
	$ npm i
	$ react-native run-ios 
	

## Fake Backend
	$ cd fake-backend
	$ npm i
